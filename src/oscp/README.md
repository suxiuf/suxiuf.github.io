OSCP （offensive security certified professional）是一个国际认可度较高的渗透测试认证，中文名“进攻性安全认证专家”，该认证更偏向于漏洞利用而不是漏洞挖掘。

# 学习方法

- 掌握理论知识，完成课后练习
- 攻击实验环境靶机，60台以上
- 练习渗透测试报告的编写
- Try Harder
    - 重复练习，确保自己熟练掌握知识点
    - 尝试独立攻击实验环境
    - 遇到问题尽量搜索、重复阅读学习资料
    - 适时寻求帮助，根据方向性的提醒，独立完成靶机
    - 做好笔记，积累经验，总结思路
    - 形成肌肉记忆
# 关于OSCP考试
- 考试不仅限于培训内容
- 考试时常：24小时，其中15分钟准备
- 70分通过考试
- 考试结束后24小时提交报告
- 10个工作日内得到成绩，成绩合格确认证书邮寄地址
