PING 命令

什么参数可以让你改变发送ping请求的时间间隔？

答案： -i

哪种交换机允许您将请求限制为IPv4？

答案： -4

哪个参数会提供更详细的输出？

答案： -v

Traceroute 命令

使用Traceroute时，您将使用哪个参数指定接口？

答案： -i

如果您想在跟踪路由时使用TCP SYN请求，您会使用哪个参数？

答案： -T

默认情况下，traceroute将在TCP/IP模型的哪一层上运行（Windows）？

答案： Internet

WHOIS

查询域名工具，也可以在浏览器上搜索🔍whois网站进行查询。

重要

在进行whois 查询时应重点关注以下字段：

  • Registered： 注册
  • Tech : 技术
  • Admin: 管理

Dig

Dig允许我们手动查询我们选择的递归DNS服务器，以获取有关域的信息：

dig <doman-name> @<DNS-server-IP>

示例:

dig www.baidu.com @223.5.5.5
